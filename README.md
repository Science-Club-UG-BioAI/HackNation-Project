# HackNation-Project
## Budget Tool - System synchronizacji i analizy danych budżetowych
Budget Tool to aplikacja webowa wspierająca proces planowania finansowego w instytucjach publicznych. Projekt powstał jako odpowiedź na rzeczywisty problem pracy z rozproszonymi danymi budżetowymi - arkuszami, dokumentami, wersjami i poprawkami przesyłanymi między wieloma pracownikami.
Celem systemu jest automatyzacja, synchronizacja oraz walidacja danych budżetowych, tak aby cały proces był prostszy, szybszy i mniej podatny na błędy.

### Frontend:
- React + JS
### Backend:
- Python + FastAPI
- SQLite
